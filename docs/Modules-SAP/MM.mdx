# SAP MM (Materials Management)

SAP Materials Management (MM) is one of the core modules in SAP that manages procurement, inventory, material planning, and vendor-related activities. It ensures that materials are available in the right quantity, at the right time, and at the right cost.

## Overview of SAP MM

SAP MM supports the complete procurement and inventory management process, from purchasing raw materials to managing stock and paying vendors.

| Feature              | Description                                    |
| -------------------- | ---------------------------------------------- |
| Procurement          | Purchasing goods and services from vendors     |
| Inventory Management | Tracking stock movements and inventory levels  |
| Material Planning    | Planning material requirements for production  |
| Vendor Management    | Managing supplier information and transactions |
| Invoice Verification | Verifying vendor invoices before payment       |

---

## Key Components of SAP MM

### Purchasing

The Purchasing component manages the procurement of materials and services from vendors.

| Activity                    | Description                                       |
| --------------------------- | ------------------------------------------------- |
| Purchase Requisition (PR)   | Internal request to procure materials or services |
| Request for Quotation (RFQ) | Request sent to vendors for pricing               |
| Quotation                   | Vendor response to an RFQ                         |
| Purchase Order (PO)         | Official order sent to the vendor                 |
| Contract                    | Long-term agreement with a vendor                 |
| Scheduling Agreement        | Delivery schedule agreement with a vendor         |

---

### Inventory Management

Inventory Management controls stock movements and maintains accurate inventory records.

| Movement Type      | Description                                               |
| ------------------ | --------------------------------------------------------- |
| Goods Receipt (GR) | Receipt of materials into inventory                       |
| Goods Issue (GI)   | Removal of materials from inventory                       |
| Transfer Posting   | Movement of stock between locations                       |
| Stock Transfer     | Transfer of materials between plants or storage locations |

---

### Material Master

The Material Master contains all information related to a material used throughout the organization.

| Data Type       | Description                            |
| --------------- | -------------------------------------- |
| Basic Data      | General information about the material |
| Purchasing Data | Procurement-related information        |
| Accounting Data | Valuation and financial information    |
| Sales Data      | Sales-related details                  |
| Plant Data      | Plant-specific information             |

---

### Vendor Master

The Vendor Master stores information about suppliers.

| Data Category     | Description                        |
| ----------------- | ---------------------------------- |
| General Data      | Name, address, and contact details |
| Company Code Data | Financial information              |
| Purchasing Data   | Procurement-related details        |

---

### Invoice Verification

Invoice Verification ensures that vendor invoices match purchase orders and goods receipts.

| Check          | Purpose                           |
| -------------- | --------------------------------- |
| Purchase Order | Verify ordered quantity and price |
| Goods Receipt  | Verify received quantity          |
| Vendor Invoice | Verify billed amount              |

---

## Procurement Process in SAP MM

| Step | Process                            |
| ---- | ---------------------------------- |
| 1    | Create Purchase Requisition (PR)   |
| 2    | Create Request for Quotation (RFQ) |
| 3    | Receive Vendor Quotations          |
| 4    | Create Purchase Order (PO)         |
| 5    | Receive Goods (GR)                 |
| 6    | Perform Invoice Verification       |
| 7    | Make Vendor Payment                |

---

## Important SAP MM Documents

| Document             | Purpose                      |
| -------------------- | ---------------------------- |
| Material Master      | Stores material information  |
| Vendor Master        | Stores vendor information    |
| Purchase Requisition | Internal procurement request |
| Purchase Order       | Formal purchasing document   |
| Goods Receipt        | Confirms receipt of goods    |
| Invoice Receipt      | Records vendor invoice       |

---

## Procure-to-Pay (P2P)

Procure-to-Pay (P2P) is the end-to-end process that starts with identifying a material requirement and ends with payment to the vendor.

### P2P Flow

1. Requirement Identification
2. Purchase Requisition
3. Vendor Selection
4. Purchase Order Creation
5. Goods Receipt
6. Invoice Verification
7. Vendor Payment

This process helps organizations efficiently manage procurement activities while maintaining accurate inventory and financial records.
