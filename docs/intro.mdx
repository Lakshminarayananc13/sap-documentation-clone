## Getting Started

- **New to SAP development?** Start with the [ABAP section](https://help.sap.com/docs/ABAP_PLATFORM_NEW/b1c834a22d05483b8a75710743b5ff26/4ec38f8788d22b90e10000000a42189d.html).
- **Working on IBP?** Head to the [IBP section](https://help.sap.com/docs/SAP_INTEGRATED_BUSINESS_PLANNING/feae3cea3cc549aaa9d9de7d363a83e6/682462bc82ed434bb45541b428268eb8.html?locale=en-US).
- **Stuck on something?** Check the [FAQ](https://support.sap.com/en/product/connectors/bc/faq.html) or raise a ticket via [SAP Launchpad](https://launchpad.support.sap.com).
